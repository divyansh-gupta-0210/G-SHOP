# G-SHOP

This is a type of APP is where I have focused mainly on UI of a E-Commerce-App, for my college.
It justs needs few details of the user, and the user can order whatever he wants from the shops inside the campus of the college.
The app is user friendly, and is pleasing to look.

# Dependencies

- cupertino_icons
- flutter_svg
- provider
- fonts "Muli"

# Contains

- Onboarding
- Login
- Forgot Password
- Sign Up
- Complete Profile
- OTP Verification
- Home Page
- Product Details
- Order
- Profile
- Bottom Navigation Bar
